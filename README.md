# Editable Resume HTML

Open [index.html](./index.html) in any browser.

Quick edits:
- Change text in `resume-data.js`.
- To add a new certificate, copy one object inside the `certificates` array.
- The `Languages` section has already been removed.
- Certificate boxes are auto-normalized to the same height on load.
- Skill bullets use one fixed square size so they stay visually consistent.

Files:
- `index.html`: page shell
- `styles.css`: design and print layout
- `resume-data.js`: all editable resume content
- `resume.js`: rendering and page flow logic
